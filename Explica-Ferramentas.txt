--------------------------------------------------------------------------------
⚠️ CASO PRÁTICO: O WINDOWS NÃO ABRE (BOOT CORROMPIDO / TELA AZUL)
--------------------------------------------------------------------------------
Se o Windows do cliente não inicia, não se preocupe! Os arquivos dele ainda estão 
guardados no HD/SSD. Veja como a ISO facilita o seu trabalho de resgate:

1. MONTAGEM AUTOMÁTICA DE DISCOS:
   Logo após a inicialização, um script varre o computador procurando os discos do
   cliente. Ele monta automaticamente as partições NTFS (Windows), FAT32 ou ext4.
   * Onde ficam os arquivos?
     Abra o Gerenciador de Arquivos (Nautilus) e vá na pasta:
     📂 /mnt/clientes/ (ex: /mnt/clientes/sda3_Windows ou sda2_Dados)
     Tudo do cliente estará lá dentro pronto para ser copiado.

2. CONTORNO DE HIBERNAÇÃO (Fast Startup / Windows Travado):
   Quando o Windows trava ou desliga incorretamente, ele deixa o HD "semi-hibernado"
   e travado contra gravação. A nossa ISO detecta isso e força a limpeza da hibernação 
   (remove_hiberfile). Assim, você consegue copiar arquivos novos para o HD dele ou
   deletar vírus diretamente.

3. MODO DE SEGURANÇA EM HDS COM DEFEITO (Bad Blocks):
   Se o HD do cliente estiver quebrado fisicamente ou morrendo, o script montará a
   partição em modo "Somente Leitura" automaticamente. Isso protege os dados do
   cliente contra novas falhas enquanto você copia o backup para o seu HD externo.

--------------------------------------------------------------------------------
1. ATALHOS NA ÁREA DE TRABALHO (SCRIPTS & INTERFACES)
--------------------------------------------------------------------------------

* [🔧 Menu Técnico EduTech]
  O painel central de controle do seu sistema. Abre uma janela gráfica (Zenity) 
  para você escolher e executar todos os scripts com facilidade usando o mouse.

* [🛡️ Scanner de Vírus Offline] (scanner-virus-offline.sh)
  Faz uma varredura completa por vírus e malwares nas partições Windows usando o ClamAV. 
  Excelente para remover malwares difíceis sem precisar ligar o Windows infectado.

* [💾 Backup de Perfil] (backup-perfil-automatico.sh)
  Copia as pastas de usuário (Área de Trabalho, Documentos, Imagens, Downloads) do 
  HD montado do cliente para um HD externo automaticamente. Usa a robustez do rsync.

* [🔑 Resetar Senha Windows] (resetar-senha-automatico.sh)
  Reseta e remove a senha de qualquer conta local do Windows. Ideal para quando o cliente
  esquece a senha de entrada da máquina.

* [💽 Diagnóstico de Discos] (diagnostico-discos.sh)
  Faz testes rápidos de hardware e exibe informações S.M.A.R.T sobre a integridade física
  do disco (setores defeituosos, horas de uso, temperatura).

* [🗃️ GParted]
  Ferramenta gráfica de partição. Use para redimensionar, formatar ou verificar 
  erros em partições do sistema.

* [📶 Conectar à Rede (SMB)] (conectar-rede.sh)
  Abre janelas interativas para conectar e montar pastas compartilhadas da sua rede local
  (Windows, NAS ou Servidor de Arquivos) na pasta /mnt/rede_local. Facilita o envio de
  backups sem precisar usar mídias físicas.

2. APLICATIVOS DO SISTEMA OPERACIONAL
--------------------------------------------------------------------------------

* [🌐 Firefox]
  Navegador web completo para pesquisar esquemas de rede ou baixar ferramentas portáteis.

* [🍷 Wine]
  Permite executar aplicativos ".exe" portáteis do Windows diretamente no Linux técnico.

* [📁 Nautilus]
  O gerenciador de arquivos visual da Área de Trabalho.

* [📊 Htop]
  Monitor gráfico do consumo de RAM, processador e temperatura em tempo real.
