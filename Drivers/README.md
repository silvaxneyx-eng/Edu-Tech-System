# Drivers offline

## Snappy Driver Installer (SDI)

Baixado automaticamente por `Scripts\Baixar-Ferramentas.ps1` em `Tools\Drivers\`.

Execute `SDI_x64.exe` ou similar — permite baixar index de drivers para uso offline.

## Pastas OEM

Coloque pacotes `.exe` / `.cab` do fabricante:

```
Drivers\
├── Intel\       ← chipset, Wi-Fi Intel
├── AMD\         ← chipset AMD
├── NVIDIA\      ← GPU NVIDIA
├── Realtek\     ← áudio/rede Realtek
└── OEM\
    ├── Dell\
    ├── HP\
    └── Lenovo\
```
