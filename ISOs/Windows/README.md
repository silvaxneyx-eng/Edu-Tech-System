# Windows — ISOs oficiais

Coloque os arquivos `.iso` nesta pasta. O Ventoy lista tudo no boot.

| Sistema | Download oficial |
|---------|------------------|
| Windows 11 | https://www.microsoft.com/software-download/windows11 |
| Windows 10 | https://www.microsoft.com/software-download/windows10 |

## Dica

Use o **Media Creation Tool** ou baixe ISO direto no site da Microsoft.
Nome sugerido: `Win11_24H2_x64.pt-BR.iso`

## Instalação

1. Boot pelo Ventoy
2. Selecione a ISO desejada
3. Instale normalmente
