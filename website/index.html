Tudo que um técnico precisa

Ferramentas automatizadas e inteligentes para agilizar seus atendimentos.

Reset de Senha Multi-SO

Detecta e remove senhas de contas locais do Windows (SAM), distribuições Linux (chroot) e até forja o recadastro de admin no macOS (.AppleSetupDone).

Detecção Inteligente

Monta automaticamente discos NTFS, FAT32, BTRFS e EXT4 ignorando partições de hibernação, pronto para você navegar como administrador.

Antivírus Offline

Fez o boot pelo pendrive? Use a engine poderosa do ClamAV para escanear partições do cliente em busca de malwares sem ligar o SO dele.

Backup de Perfil

Detecta a pasta "Users" do Windows e copia tudo o que importa (Documentos, Desktop, Fotos, Downloads) direto para o seu HD externo com um clique.

Diagnóstico de Hardware

Relatório S.M.A.R.T completo da saúde de discos rígidos e NVMe, além de puxar toda a ficha técnica do processador e memória.

Interface GTK4 Nativa

Design premium que se camufla perfeitamente no GNOME, usando as bibliotecas Adwaita com tema escuro e terminal integrado no próprio app.
